# Personal Website

## Install

This project is using [Sass](https://sass-lang.com/). To install Sass on your machine run the following command:

```
npm install -g sass
```
## Compile scss files to css

Run the following commands to compile .scss file to .css

For `styles.scss`:
```
sass scss/styles.scss css/styles.css
```

For `imprint.scss`:
```
sass scss/imprint.scss css/imprint.css
```
